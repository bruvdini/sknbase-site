# SKNBASE marketing site — screenshot shot list

Drop the PNGs into this `images/` folder using the **exact filenames** below.
The website will reference these names directly, so naming matters.

## How to capture (iPhone)
- Run the app in Expo Go, get the screen looking its best (real data, no error banners).
- Press **Side button + Volume Up** to screenshot → AirDrop / save the PNGs here.
- Keep them **portrait, full-screen, high-res** (don't crop the status bar — I'll frame them in a device mockup).
- A clean status bar looks best (full battery, no clutter) but it's optional.
